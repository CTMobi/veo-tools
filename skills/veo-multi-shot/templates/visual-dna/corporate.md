# Visual DNA: Corporate / Enterprise

A visual language preset for B2B companies, professional services, and enterprise brands.

---

## Overview

| Attribute | Value |
|-----------|-------|
| Industries | Enterprise software, consulting, financial services, healthcare, legal, B2B |
| Feeling | Trustworthy, capable, professional, reliable |
| Energy | Moderate, confident, purposeful |

---

## Locked Parameters

### Color Palette

```
PRIMARY:    Professional blues (navy to medium blue)
            #1e3a5f to #2563eb
            Trust, stability, competence

SECONDARY:  Warm neutrals (warm grays, taupe)
            #6b7280 to #9ca3af with warm undertone
            Approachable professionalism

ACCENT:     Confident accent (teal, green, or orange)
            #0d9488, #059669, or #ea580c
            Action, growth, energy (use one consistently)

SHADOWS:    Medium depth, not dramatic
            Lifted but present

HIGHLIGHTS: Clean, not blown
            Professional clarity
```

**Prompt Keywords**: `professional blue palette`, `warm neutral secondary`, `[teal/green/orange] accent`, `medium contrast`

### Lighting Style

```
QUALITY:    Soft, diffused, professional
            Studio-quality without drama

DIRECTION:  Broad, even illumination
            Nothing hidden, transparent

FILL:       Generous, open shadows
            Accessible, approachable

SPECULAR:   Controlled, subtle
            Professional finish, not flashy

ATMOSPHERE: Clean, minimal haze
            Clear communication
```

**Prompt Keywords**: `soft diffused professional lighting`, `even illumination`, `open shadows`, `clean atmosphere`

### Atmosphere / Mood

```
OVERALL:    Capable, trustworthy, forward-moving
            Gets things done

TEXTURE:    Clean, organized, purposeful
            Everything has a reason

SPACE:      Balanced composition, grounded
            Stable, not floating

FEEL:       Confident competence
            Reassuring, not aggressive
```

**Prompt Keywords**: `professional atmosphere`, `confident competence`, `organized purposeful`, `trustworthy`

### Camera Energy

```
BASELINE:   Moderate, purposeful
            Moving with intent

MOVEMENTS:  Smooth, professional, deliberate
            No wasted motion

SPEED:      Moderate, business-appropriate
            Efficient without rushing

PROGRESSION: Steady, can build toward conclusion
             Clear narrative momentum
```

**Prompt Keywords**: `purposeful camera movement`, `moderate pace`, `deliberate smooth motion`

```
[Smooth camera movement] [following/through] [abstract process visualization],
[organized flowing elements], clear progression,
professional blue palette with [accent] highlights, soft diffused lighting,
professional atmosphere, purposeful motion, [process descriptor]
```

### Solution / Product Shot
```
[Camera movement] [toward/around] [service/product visualization],
[organized, functional forms], clear purpose,
professional blue palette with warm neutral secondary, even illumination,
professional atmosphere, confident capability, [value proposition]
```

```
[Static / slow movement], [abstract data visualization],
[clean organized information flow], clarity and structure,
professional blue palette with [accent], soft diffused lighting,
professional atmosphere, transparent and trustworthy, [information quality]
```

```
[Pull out / expanding movement] revealing [success visualization],
[growth or achievement representation], sense of accomplishment,
professional blue palette with [accent] highlights, warm optimistic lighting,

```

## Industry-Specific Variations

### Financial Services
- More conservative blues
- Minimal accent usage
- Add: `stable`, `secure`, `established`

### Healthcare / Life Sciences
- Warmer blues, more white space
- Green or teal accents
- Add: `caring`, `precise`, `life-improving`

### Technology / Enterprise Software
- Can be more dynamic
- Bolder accent usage
- Add: `innovative`, `efficient`, `powerful`

### Consulting / Professional Services
- Warmer neutrals
- Sophisticated but approachable
- Add: `expert`, `partnership`, `guidance`

---

## Variations

### Innovative Corporate
- More dynamic camera
- Bolder accents
- Add: `forward-thinking`, `innovative`, `transforming`

### Traditional Corporate
- More static shots
- Muted accents
- Add: `established`, `reliable`, `trusted`

### Human-Centered Corporate
- Warmer overall palette
- More organic forms
- Add: `people-focused`, `empowering`, `supportive`

## Abstract Visualization Guide

Since corporate videos often visualize abstract concepts, here are effective metaphors:

### Positive Concepts

| Concept | Visual Approach |
|---------|-----------------|
| Growth | Expanding forms, upward motion, branching |
| Efficiency | Streamlined flows, direct paths, smooth motion |
| Connection | Linking nodes, converging paths, networks |
| Security | Contained structures, protective forms, solid geometry |
| Innovation | Emerging forms, transformation, light reveals |
| Collaboration | Merging elements, synchronized motion, unity |

### Transformation Journeys

| From | To | Visual Transition |
|------|----|--------------------|
| Chaos | Order | Scattered → Aligned |
| Slow | Fast | Hesitant → Flowing |
| Isolated | Connected | Separate → Linked |
| Complex | Simple | Tangled → Clean |
| Old | New | Faded → Vibrant |

## Continuity Checklist

When using this Visual DNA across multiple shots, verify:

- [ ] "professional blue palette" or equivalent in ALL prompts
- [ ] "soft diffused lighting" or "even illumination" in ALL prompts
- [ ] "professional atmosphere" or equivalent in ALL prompts
- [ ] Accent color is consistent (same color in all prompts)
- [ ] Camera energy is consistently purposeful/moderate
- [ ] No prompt feels too dramatic or too casual
- [ ] Abstract visualizations are consistently professional (not playful)
