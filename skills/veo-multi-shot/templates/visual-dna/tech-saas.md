# Visual DNA: Tech / SaaS

A visual language preset for technology companies, software products, and digital services.

---

## Overview

| Attribute | Value |
|-----------|-------|
| Industries | SaaS, fintech, dev tools, AI/ML, cloud services |
| Feeling | Innovative, trustworthy, forward-thinking |
| Energy | Measured to moderate, professional precision |

---

## Locked Parameters

### Color Palette

```
PRIMARY:    Cool blue (#1e3a5f to #3b82f6 range)
            Deep tech blue, suggests trust and innovation

SECONDARY:  Slate gray (#475569 to #94a3b8 range)
            Neutral, professional, supports without competing

ACCENT:     Warm amber/gold (#f59e0b) OR Electric cyan (#06b6d4)
            Sparingly used, draws attention to key moments

SHADOWS:    Lifted blacks, never pure black
            Maintains ethereal, modern feeling

HIGHLIGHTS: Soft whites, never blown out
            Clean, digital precision
```

**Prompt Keywords**: `cool blue palette`, `slate gray secondary`, `warm amber accent` or `cyan accent`, `lifted blacks`

### Lighting Style

```
QUALITY:    Soft, diffused
            Clean edges without harshness

DIRECTION:  Side-lit with subtle rim separation
            Defines form without drama

FILL:       Low contrast, open shadows
            Accessible, nothing hidden

SPECULAR:   Subtle, controlled highlights
            Premium without flashy

ATMOSPHERE: Subtle depth haze, particle-rich
            Adds dimension, tech ambiance
```

**Prompt Keywords**: `soft diffused lighting`, `subtle rim separation`, `low contrast`, `soft atmospheric haze`

### Atmosphere / Mood

```
OVERALL:    Ethereal, innovative, aspirational
            Future-forward but approachable

TEXTURE:    Floating particles, subtle data visualization
            Tech-native visual language

SPACE:      Clean, organized, intentional negative space
            Clarity and purpose

FEEL:       Calm confidence, quiet innovation
            Trustworthy, not aggressive
```

**Prompt Keywords**: `ethereal tech atmosphere`, `floating particles`, `clean composition`, `calm innovation`

### Camera Energy

```
BASELINE:   Measured, contemplative
            Professional pacing

MOVEMENTS:  Smooth, continuous, never jerky
            Precision in motion

SPEED:      Slow to moderate
            Confidence, not urgency

PROGRESSION: Can increase slightly toward climactic moments
             Build toward insight or reveal
```

**Prompt Keywords**: `smooth measured movement`, `contemplative pace`, `controlled precision`

## Variable Parameters (Per Shot)

### Camera Movement Options

| Movement | Best For | Energy |
|----------|----------|--------|
| Slow dolly forward | Approaching, discovery | Low-Med |
| Push in | Focus, emphasis, reveal | Medium |
| Pull out | Context, expansion | Medium |
| Slow orbit | Product showcase, examination | Low-Med |
| Static | Data visualization, loops | Low |
| Tracking | Journey, process | Medium |

### Shot Size Options

| Size | Best For |
|------|----------|
| Wide | Establishing environment, data landscapes |
| Medium | Product in context, clear subject |
| Close-up | Detail, material, UI elements |
| Macro | Texture, premium detail |

```
[Camera movement] through [tech environment descriptor],
[floating particles / data visualization elements],
cool blue palette with warm amber accents, soft diffused lighting,
subtle rim separation, ethereal tech atmosphere, [shot-specific mood]
```

```
[Camera movement] [toward/around] [product/feature description],
[specific material or interface detail],
cool blue palette with warm amber accents, soft diffused lighting,
ethereal tech atmosphere, [product emotion: innovation/precision/clarity]
```

```
[Slow orbit / static], macro view of [specific tech surface/element],
[light behavior on surface],
cool blue reflections with amber accent, soft diffused key light,
```

```
[Pull out / crane up] revealing [expanded tech landscape/system],

ethereal tech atmosphere elevated, [positive emotion]
```

## Variations

### Warmer Tech (Startup, Consumer)
- Increase amber accent usage
- Softer, more approachable lighting
- Add: `approachable`, `friendly`, `warm tech`

### Cooler Tech (Enterprise, Security)
- Minimize warm accents
- More cyan accents if needed
- Add: `secure`, `enterprise`, `institutional`

### Premium Tech (High-end, Luxury Tech)
- Deeper blues, more contrast
- Subtle gold accents
- Add: `premium`, `exclusive`, `refined`

- [ ] "cool blue palette" or equivalent appears in ALL prompts

- [ ] "ethereal tech atmosphere" or equivalent appears in ALL prompts
