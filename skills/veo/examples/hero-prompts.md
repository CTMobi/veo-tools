## Quick Customization Guide

### Swap Elements

**Change mood** - Replace atmosphere/color descriptors:
- Ethereal → Industrial
- Warm → Cool
- Serene → Dramatic

**Change pace** - Adjust motion descriptors:
- "slowly" → "imperceptibly" (more subtle)
- "gently" → "continuously" (more active)

**Change subject** - Keep structure, replace focal element:
- "data particles" → "bioluminescent spores"
- "concrete wall" → "marble surface"
- "forest canopy" → "underwater kelp forest"

### Essential Loop Flags

Always include for hero backgrounds:
```
seamless loop, locked camera
```

For extra subtlety:
```
subtle motion, gentle movement
```

### Duration Guidance

- **4 seconds**: Simple motion, single element
- **6 seconds**: Complex scenes, environmental
- **8 seconds**: Only if motion truly needs it (rare for heroes)

## Prompt Template

```
[Camera: locked/static] [Composition: camera angle or framing]
[Subject: specific material/form with detail]
[Motion: present continuous, subtle verb]
[Environment: location context if needed]
[Lighting: source and quality]
[Color: palette direction]
seamless loop, [additional mood descriptors]
```

### Template Application Example

**Requirement**: Healthcare/wellness SaaS hero background

**Apply template**:
- Camera: Static, centered
- Subject: Soft organic forms (not clinical)
- Motion: Breathing rhythm, gentle expansion
- Lighting: Soft, warm, diffused
- Color: Muted earth tones, calming

**Result**:
```
Static centered camera, soft organic cellular forms gently expanding and contracting in breathing rhythm, warm diffused ambient light, muted earth tone palette with soft cream highlights, seamless loop, calming wellness atmosphere
```
